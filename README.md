# Heritage Health Services — HeritageHealthNew

Premium personal care agency website serving Southeast Wisconsin. This project features a high-end, medical-grade aesthetic focused on trust, professionalism, and compassionate care.

## 🚀 Getting Started

To run this site locally:

1. **Navigate to the directory:**
   ```bash
   cd Desktop/HeritageHealthNew
   ```

2. **Start a local server:**
   Using `serve`:
   ```bash
   npx serve -l 5175
   ```
   Or using Python:
   ```bash
   python3 -m http.server 5175
   ```

3. **Open in Browser:**
   Visit `http://localhost:5175`

## 📁 Project Structure

- `index.html`: Main landing page with all sections (Hero, Services, About, Assessment, etc.)
- `portal.html`: Client/Employee portal layout
- `style.css`: Core Design System and component styling
- `main.js`: Interactive elements, scroll animations, and chatbot logic
- `assets/`: Image assets (caregiver photos, icons, and branding)

## 🎨 Design Philosophy

The design leverages a **Navy and Gold** palette to evoke a sense of heritage and premium quality. 
- **Typography:** Uses 'Playfair Display' for an elegant, serif look in headings and 'Inter' for highly readable body text.
- **Motion:** Integrated "Reveal on Scroll" system for a modern, fluid user experience.
- **Aesthetic:** "Medical-grade refined"—clean white spaces balanced with deep navy sections and gold accents.

## 🛠 Features

- **Responsive Design:** Fully optimized for mobile, tablet, and desktop.
- **Scroll Animations:** Smooth entry for content sections.
- **Integrated Chatbot:** Support bubble for quick inquiries.
- **Assessment Form:** Professional enrollment flow for new clients.
- **Heritage Connect:** A conceptual portal entry for clinical transparency.
