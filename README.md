# Heritage Health Services — HeritageHealthNew

[![Live Site](https://img.shields.io/badge/Live-Site-0A1F3A?style=for-the-badge&logo=github&logoColor=C9A84C)](https://xibalbatechsol.github.io/HeritageHealthNew/)

> **"How Personal Care Should Be."**  
> Heritage Health Services is Southeast Wisconsin’s premier RN-led personal care agency, specializing in Medicaid (Title 19) and private pay home care.

## 🌟 Overview

HeritageHealthNew is a high-end, medical-grade website designed to bridge the gap between clinical excellence and the warmth of home. The platform is built with a focus on trust, transparency, and the unique Wisconsin benefit that allows families to choose and get paid to care for their loved ones.

## 🚀 Key Features

- **RN-Led Clinical Oversight:** Emphasizing safety and professional standards in a home setting.
- **"Choose Your Caregiver" Flow:** Dedicated sections explaining how family members can get paid to provide care through Medicaid.
- **Premium Design System:** A sophisticated Navy and Gold palette designed for a "medical-grade refined" aesthetic.
- **Interactive Experience:** 
    - Smooth scroll animations using Intersection Observer.
    - Custom-built interactive Chatbot UI.
    - Integrated Patient/Employee portal preview (Heritage Connect).
- **Responsive & Accessible:** Fully optimized for all devices, from desktop assessments to mobile inquiries.

## 🛠 Tech Stack

- **Frontend:** HTML5, Vanilla CSS3 (Custom Properties, CSS Grid, Flexbox).
- **Interactions:** Vanilla JavaScript (ES6+).
- **Animations:** Custom "Reveal on Scroll" logic for a fluid, high-end feel.
- **Icons:** FontAwesome 6 Pro.
- **Typography:** 
    - `Playfair Display`: Used for elegant, authoritative headings.
    - `Inter`: Used for crisp, accessible body text.

## 🎨 Design Philosophy

The site follows a strict **"Medical-Grade Refined"** philosophy:
- **Navy (#0A1F3A):** Represents authority, professionalism, and heritage.
- **Gold (#C9A84C):** Represents the "gold standard" of care and premium service.
- **Off-White (#FAFAF8):** Provides a clean, clinical yet warm background.
- **Whitespace:** Generous use of padding to maintain a calm, premium browsing experience.

## 🔧 Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/XibalbaTechSol/HeritageHealthNew.git
   ```
2. **Launch a local server:**
   ```bash
   # Using Python
   python3 -m http.server 8000
   ```
3. **Open in Browser:** `http://localhost:8000`
